# Word Frequency Counter (Python)

This is a simple Python project that counts how many times each word appears in a sentence.

Features
- Converts text to lowercase
- Removes punctuation
- Splits sentence into words
- Counts word frequency using dictionary

 Example

**Input:**
AI is the future and AI is powerful

**Output:**
ai : 2  
is : 2  
the : 1  
future : 1  
and : 1  
powerful : 1  

Concepts Used
- Strings
- Lists
- Dictionaries
- Loops
